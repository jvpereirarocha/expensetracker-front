# SKILL: Application Style Guide

# CONTEXT
This is the visual language for the "Financial Tracker" application. Use these Tailwind classes strictly.

## 1. Color Palette (Financial)
- **Primary Action:** `bg-indigo-600 hover:bg-indigo-700 text-white`
- **Secondary/Outline:** `bg-white border border-slate-300 text-slate-700 hover:bg-slate-50`
- **Danger/Delete:** `bg-rose-600 hover:bg-rose-700 text-white`

## 2. Financial Indicators (STRICT)
- **Income/Positive:** - Text: `text-emerald-600`
  - Background: `bg-emerald-50`
  - Border: `border-emerald-200`
- **Expense/Negative:** - Text: `text-rose-600`
  - Background: `bg-rose-50`
  - Border: `border-rose-200`

## 3. Spacing & Layout
- **Page Container:** `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-8`
- **Card:** `bg-white shadow-sm border border-slate-200 rounded-xl p-6`
- **Grid:** `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6`

## 4. Typography
- **H1 (Page Title):** `text-2xl font-bold leading-7 text-gray-900 sm:text-3xl`
- **H2 (Section):** `text-lg font-medium leading-6 text-gray-900`
- **Body:** `text-sm text-slate-600`
